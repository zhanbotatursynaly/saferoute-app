# SafeRoute 🚀

SafeRoute is a mobile application that helps users travel more safely in the city by identifying dangerous areas and suggesting safer routes instead of only the shortest path.

---

## 💡 Problem

Many people feel unsafe when walking alone, especially at night.  
Existing navigation apps like Google Maps focus on the fastest route, not the safest one.

As a result:
- Users avoid streets based on guesswork  
- Safety decisions are emotional, not data-driven  
- There is no tool focused on personal safety navigation  

---

## ✅ Solution

SafeRoute provides:
- A map with dangerous areas (reported by users)
- Safer route suggestions based on risk levels
- A simple way to report unsafe locations

👉 “We optimize for safety, not speed.”

---

## 🎯 Target Users

- Female students (17–25)
- People walking alone at night
- Urban residents

---

## 🔥 Key Features

- 🔐 Authentication (Email & Google Sign-In)
- 🗺️ Map with danger points (Firebase Firestore)
- 🛣️ Safer route suggestions
- ➕ Report unsafe places
- 📊 Real-time updates

---

## ⚙️ Tech Stack

- Flutter (UI)
- Firebase Authentication
- Cloud Firestore
- Google Maps API
- Geolocator

---

## 🧪 MVP Scope

This project is an MVP (Minimum Viable Product) focused only on core features:
- User authentication
- Danger map visualization

- Reporting system

## 💰 Business Model (Future)

## 📊 Validation (Sample)

- 5 out of 6 students reported feeling unsafe at night  
- Most users said they would use a safety-based navigation app  
- Users prefer a safer route even if it is longer
